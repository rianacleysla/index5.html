Propriedades CSS

O CSS tem várias propriedades que nos deixa controlar a aparência dos elementos em uma página. Algumas das propriedades comuns são:

Propriedade Descrição Exemplo
color Define a cor do texto. color: red;
font-family Define a fonte do texto. font-family: Arial, sans-serif;
margin Controla o espaço fora de um elemento. margin: 20px;
font-size Controla o tamanho da fonte. font-size: 10px;
background-color Define a cor de fundo de um elemento. background-color: #f0f0f0;
borders Define a margem da tabela. margin;

Essas propriedades ajudam a personalizar suas páginas web.
