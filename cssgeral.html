CSS

CSS (Cascading Style Sheets) é uma linguagem para estilo usada para "decorar"documentos HTML. O CSS nos deixa mudar a aparência de elementos em uma página HTML, como cores, fontes, tamanhos e layouts. A estrutura do CSS é formada por seletores, que selecionam os elementos a serem estilizados.

O CSS também permite a adição de animações e transições, melhorando a interatividade e a estética das páginas.

O termo "cascading" se refere à maneira como o CSS aplica estilos, resolvendo conflitos com base na especificidade e na ordem das regras. Com media queries, é possível adaptar o design a diferentes tamanhos de tela, tornando as páginas mais acessíveis em diversos dispositivos.
