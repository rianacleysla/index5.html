Seletores CSS

Os seletores CSS são usados para selecionar os elementos HTML para serem estilizados. Alguns seletores são:

  • Seletor de classe: Permite que o estilo seja aplicado a diversos elementos. Exemplo: .minha-classe seleciona todos os elementos com a classe "minha-classe".
  • Seletor de ID: Seleciona um único elemento com um ID específico. Exemplo: #meu-id seleciona o elemento com ID "meu-id".

Os seletores permitem um controle sobre qual elemento será estilizado na página.
